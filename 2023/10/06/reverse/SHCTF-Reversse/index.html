  1. cdqe:

    • The cdqe instruction stands for “Convert Doubleword to Quadword, Extend with Sign.”
    • In simpler terms, it extends a 32-bit value (a doubleword) to a 64-bit value (a quadword) by copying the sign bit (the highest bit) of the 32-bit value into all the higher-order bits of the 64-bit result.
    • Specifically, it takes the value in the eax register, extends it to 64 bits, and places the result in the rax register.

  2. mov [rax+rdx], cl:

    • This mov instruction is moving a byte of data from one place to another.
    • The source is the cl register, which is the lowest byte of the rcx register.
    • The destination is a memory location, whose address is computed by adding the values of the rax and rdx registers.
    • In this instruction, the mov instruction is storing the byte from cl into the memory location whose address is the sum of rax and rdx.
