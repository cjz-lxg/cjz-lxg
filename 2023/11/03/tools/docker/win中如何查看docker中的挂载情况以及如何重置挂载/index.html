win中如何查看docker中的挂载情况以及如何重置挂载

发表于2023-11-03|更新于2023-11-09|tools
|字数总计:199|阅读时长:1分钟|阅读量:

背景

这篇博客的起因主要是因为我自己经常写docker-compose.yml来配置开发环境,解决的是我自己提前设置nacos时他一直不按照我写的目录来配置映射目录的问题过程中的解决问题的思路

步骤

查看挂载情况

这里以查看nacos为例
首先打开docker desktop ,点击下图画圈位置

点击下图红圈部分即可查看

重置挂载

首先就是在docker-compose.yml 中写下要重新挂载的目录位置, 然后点击下面的画红圈的地方先删除当前容器,不删除就挂载不上去,当你删除成功,你最开始如何启动这个容器的几句怎么重新启动就好
