前言

本篇文章将简要介绍下java的内部类

静态内部类

顾名思义, 这个内部类可以定义静态方法与静态成员变量 , 同时也可以访问外部类的静态成员变量与方法, 但是无法访问普通成员变量与方法,其实也很好理解, 同时, 需要注意, 外部可以通过下面的方式访问内部类的静态成员变量与方法, 但是访问不了普通成员变量与方法

成员内部类

其与静态内部类最大的区别就是无法访问外部类的静态方法与成员变量, 同时自己也无法定义这些,但是外部可以通过创建外部类的实例对象来访问在其中定义的普通方法

方法内部类

我几乎不使用, 也就是定义在方法中的类

匿名内部类

常见于Lamba表达式中, 只能实现一个接口或者继承一个类, 这东西平常用的最多.
