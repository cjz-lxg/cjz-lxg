使用

The Properties class in Java is a subclass of Hashtable. It is used to maintain lists of values in which the key is a String and the value is also a String1. The Properties class represents a persistent set of properties, which can be saved to a stream or loaded from a stream2. Each key and its corresponding value in the property list is a string2.

Here are some of the key methods provided by the Properties class3:

  • load(Reader r): Loads data from the Reader object.
  • load(InputStream is): Loads data from the InputStream object.
  • getProperty(String key): Returns value based on the key.
  • setProperty(String key, String value): Calls the put method of Hashtable.
  • list(PrintStream out): Prints the property list out to the specified output stream.
  • propertyNames(): Returns an enumeration of all the keys from the property list.
  • stringPropertyNames(): Returns a set of keys in from property list where the key and its corresponding value are strings.
  • store(Writer w, String comment): Writes the properties in the writer object.
  • store(OutputStream os, String comment): Writes the properties in the OutputStream object.
